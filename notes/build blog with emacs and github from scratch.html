Build Blog With Emacs and Github from Scratch

Table of Contents

  • 1. 创建 github 仓库
  • 2. 参考资料

本博客使用 Emacs 中自带的 ox-publish 模块将 org 文件转成静态 html 文件，然后将 html 文件上传到 github 进行托管，使用 github pages 功能来访问网页。所以本博客需要使用到的功具有：

  • Emacs + org-mode
  • Github + github pages

此种方式建立的博客比较容易，成本很低。但是博客的样式需要自己编写 CSS 进行定制。
